# Cybersecurity-Themed Portfolio Website

A cutting-edge portfolio website designed for a cybersecurity and Python developer, featuring advanced glitch effects, security-themed animations, and interactive elements that create an immersive cybersecurity aesthetic.

## Features

### Advanced Glitch Effects
- RGB-split glitch effect for main headings
- Terminal-style text animation with glitching
- Hover glitch effects for navigation elements
- Cyber-themed glitch effects for section headings
- Text corruption effects

### Security-Themed Visual Elements
- Animated security badge
- Project card hover effects with glitch animation
- Scanner animations for project cards and sections
- Code snippet with typing animation
- Secure form validation with encryption animation

### Interactive Cybersecurity Features
- Matrix-style digital rain animation (runs every 10 seconds)
- Cybersecurity-themed form validation
- "Encryption" animation for form submission
- Error handlers with glitch animations
- Security status indicator that changes with scroll depth

### Enhanced Performance
- Cross-browser compatibility fixes
- Mobile optimization
- Reduced motion support
- Progressive image loading
- Performance monitoring

# Portfolio Project with Flask

## Structure
- `app.py`: Main Flask application
- `assets/`: Additional resources (images, docs, etc.)
- `instance/`: Instance-specific config/data
- `static/`: Static files (CSS, JS, images)
- `templates/`: HTML templates

## How to Run
1. Install dependencies: `pip install flask`
2. Start the server:
   ```powershell
   python app.py
   ```
3. Visit [http://127.0.0.1:5000](http://127.0.0.1:5000)
