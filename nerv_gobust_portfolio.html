At a Glance

What is go_bust?

go_bust is a fully featured Python automation wrapper around the Gobuster enumeration tool. Rather than manually crafting repetitive scan commands, go_bust handles target discovery, multi-mode scanning, wildcard response filtering, and professional reporting — all from a single, clean CLI interface.

Capabilities

Core Features

Every feature was built to solve a real problem encountered during live penetration testing engagements and CTF challenges.

🔍

Automated Port Discovery

Uses raw TCP socket connections to scan 14 common web ports before enumeration begins, automatically building scan targets from discovered services.

  • Ports 80, 443, 8080, 8000, 8888, 3000, 5000 and more
  • Auto-selects HTTP vs HTTPS per port
  • Configurable socket timeout
  • Live progress indicator during discovery
📁

Multi-Mode Directory Enumeration

Wraps Gobuster's dir mode with intelligent defaults, extension lists, and all flags exposed via a clean CLI — no manual command crafting needed.

  • Quick scan (basic dirs) and full extended scan (13 file types)
  • Custom blacklists, user-agents, and proxy support
  • Wildcard response filtering via --exclude-length
  • Output saved per-scan with timestamps
🌐

DNS Subdomain & VHost Enumeration

Automates Gobuster's DNS and vhost modes for complete external attack surface mapping.

  • Subdomain brute-force against live DNS resolvers
  • Virtual host enumeration for shared hosting environments
  • Domain-aware scan routing
  • Supports SecLists DNS wordlists automatically
🛡️

Exploit Analysis Engine

After each scan, results are automatically analysed against 8 vulnerability pattern categories with actionable exploitation guidance.

  • Risk-rated: Critical / High / Medium
  • Sample payloads per vulnerability type
  • OWASP Top 10 & CWE references included
  • Standalone analyze command for existing result files
🎯

Intelligent Wordlist Selection

The tool scans the local filesystem for installed wordlist packages and automatically selects the best match for each scan type.

  • Supports SecLists, DirBuster, DIRB, Wfuzz, FuzzDB
  • Separate priority lists for directory, DNS, API, and quick scans
  • Graceful fallback with install guidance on missing wordlists
📊

Professional Reporting

Every scan generates structured, timestamped output files with clean ANSI-stripped content suitable for documentation and reporting.

  • Summary reports with statistics per scan
  • Visual directory tree of discovered structure
  • Dedicated exploit recommendation files
  • All results saved to gobuster_results/
⚡

Wildcard Response Handling

Detects and gracefully handles servers that return false positives by catching Gobuster's wildcard error and surfacing actionable guidance.

  • --exclude-length to filter by response size
  • Custom status code blacklisting
  • Clear error messages with fix instructions
🎨

Custom Terminal UI

Features a branded GO_BUST ASCII art banner with full ANSI colour support on Linux terminals and animated real-time progress indicators.

  • Threaded spinner with elapsed time counter
  • Colour-coded output (red / cyan / yellow / green)
  • Unicode box-drawing fallback for limited terminals
  • Windows colour enable via os.system('color')

CLI Interface

Scan Modes

Seven subcommands cover every phase of web reconnaissance, from initial port discovery to deep content enumeration.

auto
TCP port scan then auto-enumerate all discovered web services
go_bust.py auto -t 10.10.10.100
quick
2-phase rapid dir scan with and without file extensions
go_bust.py quick -u http://target
full
4-phase comprehensive scan including DNS and vhost
go_bust.py full -u http://target -d example.com
dir
Manual directory brute-force with full flag control
go_bust.py dir -u http://target -w wordlist.txt
dns
Subdomain enumeration via DNS brute-force
go_bust.py dns -d example.com
vhost
Virtual host discovery on shared-hosting targets
go_bust.py vhost -u http://target -d example.com
analyze
Standalone exploit analysis on any existing result file
go_bust.py analyze -f results.txt -u http://target
